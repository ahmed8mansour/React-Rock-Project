Welcome to my React Portfolio – a stunning showcase of my work built with React.js.

## ✨ Features
- Modern UI with animations
- Fully responsive
- Project showcase
- Contact form

## 🛠 Tech Stack
- html
- CSS
- bootstrap
- js
- react
- redux toolkit


## 🚀 Quick Start
1. Clone the repo:
   ```sh
   git clone https://github.com/ahmed8mansour/React-Rock-Project.git
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Run dev server:
   ```sh
   npm run dev
   ```
